# Getting Started

### Clone this repository

```
git clone https://github.com/viniciusdacal/tap-game
```

### Install dependencies

with yarn

```
yarn install
```

with npm

```
npm install
```

### Run

```
  yarn start
```

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
